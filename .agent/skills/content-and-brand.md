## 1. Brand Voice

### Positioning Template
```
For [TARGET AUDIENCE] who [NEED/DESIRE],
[PRODUCT] is a [CATEGORY] that [KEY DIFFERENTIATOR].
Unlike [ALTERNATIVES], we [UNIQUE VALUE].
```

### Voice Attributes
Define 3-5 voice attributes per project. Example format:

| Attribute | Means | Does NOT Mean |
|---|---|---|
| [e.g., Professional] | [e.g., Evidence-based, structured] | [e.g., Cold, academic jargon] |
| [e.g., Approachable] | [e.g., Clear language, guiding] | [e.g., Dumbed-down, patronizing] |

## 2. Content Strategy

### Content Types Matrix
| Type | Purpose | Frequency | SEO Value |
|---|---|---|---|
| Feature pages | Conversion | Per feature | High |
| Comparison pages | SEO, conversion | Quarterly | High |
| How-to guides | Engagement, SEO | Monthly | Medium |
| Blog articles | SEO, thought leadership | Monthly | Medium |
| Changelog | Retention | Per release | Low |

### Content Lifecycle
Plan -> Create -> Optimize -> Publish -> Measure -> Iterate

## 3. SEO Copywriting

### On-Page Essentials
| Element | Rule |
|---|---|
| **Title tag** | <=60 chars, primary keyword first, brand last |
| **Meta description** | <=155 chars, include primary keyword, call-to-action |
| **H1** | One per page, matches search intent |
| **URL slug** | Lowercase, hyphens, 3-5 words max |
| **Alt text** | Descriptive, keyword-natural, <=125 chars |

### Keyword Strategy
1. **Primary keyword:** The main search term for the page
2. **Secondary keywords:** 2-3 related terms woven naturally
3. **Long-tail:** Question-based phrases for FAQ/guide content

### Content Quality Signals
- Every claim backed by data or source
- Internal links to related pages
- External links to authoritative sources
- Updated date visible

## 4. Content Marketing

### Distribution Checklist
| Channel | Content Type | When |
|---|---|---|
| Blog/website | Long-form, guides | Ongoing |
| Social media | Snippets, teasers | Launch + weekly |
| Email | Updates, tutorials | Monthly |
| Forums/communities | Answers, discussions | Ongoing |

### Writing Rules
1. Lead with the value proposition, not features
2. Use data and specifics, not vague superlatives
3. Match the audience's language and terminology
4. Every piece has a clear call-to-action
