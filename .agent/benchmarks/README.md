# Skill Benchmarking

This directory tracks the effectiveness of agent skills over time.

## How to Benchmark

Use the `/skill-benchmark` approach: run a task with and without a specific skill enabled, then compare outcomes.

## Tracking Template

| Date | Skill Tested | Task | Without Skill | With Skill | Improvement |
|---|---|---|---|---|---|
| YYYY-MM-DD | `code-review-excellence` | PR review | Missed 3 issues | Caught all issues | +100% detection |

## Key Metrics

- **Accuracy:** Did the skill produce correct output?
- **Token Efficiency:** Did the skill reduce total tokens consumed?
- **Time:** Did the skill reduce task completion time?
- **Quality:** Did the skill improve output quality (subjective 1-10)?
